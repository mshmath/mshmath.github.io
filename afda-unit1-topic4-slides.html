Comparing Data Patterns

Adding Pattern (+)

Linear functions grow by adding the same amount each time.

2, 4, 6, 8, 10...
(+2 each step)

Multiplying Pattern (×)

Exponential functions grow by multiplying by the same amount each time.

2, 4, 8, 16, 32...
(×2 each step)

Who Wins the Race?

In the long run (as $x \to \infty$):

Exponential > Quadratic > Linear

Exponential growth will eventually overtake everything else.

Keywords to Look For

Linear Constant rate, Per, Every, Flat fee
Quadratic Projectile, Thrown ball, Max height, Gravity
Exponential Doubles, Triples, Percent growth, Population
