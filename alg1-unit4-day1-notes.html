I Do: What is a System?

A system of equations is two or more lines on the same graph. The solution is the one point where the lines cross.

$y = x+1$
$y = -2x+4$

Method 1: Graphing

Method 2: Algebra

Since both equations equal $y$, we can set them equal to each other.

Step 1: Set equations equal.

$x+1 = -2x+4$

Step 2: Solve for $x$.

$3x+1=4 \Rightarrow 3x=3 \Rightarrow x=1$

Step 3: Plug $x$ back in to find $y$.

$y = (1) + 1 \Rightarrow y=2$

Step 4: Write the solution.

The solution is $(1,2)$.

We Do: Finding the Solution

Let's find the solution for this system together.

$y = -x + 5$
$y = \frac{1}{2}x + 2$

  1. Graphically: Type both equations into Desmos. Click the intersection point. What are the coordinates?
  2. Algebraically: Now, let's check it by hand. Set the equations equal to each other and solve: $-x+5 = \frac{1}{2}x + 2$.
