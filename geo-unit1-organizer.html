Geometry: Unit 1

Logic, Reasoning & Venn Diagrams

VA SOL Standards: G.1a, G.1b, G.1c
Essential Question "How can we use logic and diagrams to determine the validity of an argument?"
1. Conditional Statements (SOL G.1a)

Original Statement: If hypothesis (p), then conclusion (q).

Form Symbol Rule & Meaning
Conditional $p \rightarrow q$ If $p$, then $q$. (Original)
Converse $q \rightarrow p$ FLIP the order.
If $q$, then $p$.
Inverse $\sim p \rightarrow \sim q$ NEGATE both.
If not $p$, then not $q$.
Contrapositive $\sim q \rightarrow \sim p$ FLIP & NEGATE.
If not $q$, then not $p$. (Equivalent to Original!)
Biconditional $p \leftrightarrow q$ Original & Converse are both True.
"$p$ if and only if $q$" (iff)
2. Deductive Reasoning (SOL G.1b)

Law of Detachment

If $p \rightarrow q$ is true,
And $p$ happens (is true),
Then $q$ is true.

Law of Syllogism (Chain Rule)

If $p \rightarrow q$ is true,
And $q \rightarrow r$ is true,
Then $p \rightarrow r$ is true.
3. Venn Diagrams & Sets (SOL G.1c)
Term Symbol Visual / Meaning
Union $A \cup B$ "A OR B" (Everything in both circles)
Shade All
Intersection $A \cap B$ "A AND B" (Only the overlap)
Shade Middle
Complement $\sim A$ "NOT A" (Everything outside A)
Shade Outside Circle
Key Vocabulary
Inductive Reasoning Using patterns or specific examples to make a conjecture (guess).
Deductive Reasoning Using facts, definitions, and logic laws to prove a conclusion.
Counterexample One specific example that proves a statement is FALSE.
Logically Equivalent Statements that always have the same truth value (e.g., Conditional & Contrapositive).
Subset When every element of Set A is also in Set B (Circle A is inside Circle B).
Disjoint Sets Sets that have no elements in common (Circles do not touch).
