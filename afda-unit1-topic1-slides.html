Parent Functions

The 6 Key Families

SOL AF.1.a

Objectives & Standards

Today's Goals:

  • Identify the graphs and equations of 6 Parent Functions.
  • Define key features: Domain, Range, Vertex, Asymptote, and Turning Point.
  • Recognize shapes: Linear, Quadratic, Exponential, Absolute Value, Square Root, Cubic.

What is a Parent Function?

"The Source"

Definition:

The simplest form of a function family.

It has no transformations (no shifts, no stretches, no flips).

All other graphs in the family are just modified versions of this one.

Anatomy of a Graph

Inputs & Outputs

Domain (Inputs: x)
All possible x-values. Read graph Left to Right.

Range (Outputs: y)
All possible y-values. Read graph Bottom to Top.

Shape Changers

Vertex
A sharp turning point (Maximum or Minimum).

Asymptote
A dashed line the graph gets closer to but never touches.

Turning Point
Where a curve changes direction (e.g. from curving up to curving down).

1. Linear Family

Straight Line

y = x

Domain: All Real Numbers
Range: All Real Numbers
Key Point: (0, 0) Origin
Feature: Constant Slope

2. Quadratic Family

Parabola (U-Shape)

y = x2

Domain: All Real Numbers
Range: $y \ge 0$
Vertex: (0, 0)
Feature: Symmetric

3. Exponential Family

Exponential Curve

y = 2x

Domain: All Real Numbers
Range: $y > 0$
Asymptote: $y = 0$ (x-axis)
Feature: Rapid Growth

4. Absolute Value

V-Shape

y = |x|

Domain: All Real Numbers
Range: $y \ge 0$
Vertex: (0, 0)
Feature: Sharp Turn

5. Square Root

Arc / Eyebrow

y = √x

Domain: $x \ge 0$
Range: $y \ge 0$
Start Point: (0, 0)
Feature: One direction

6. Cubic Family

S-Curve / Snake

y = x3

Domain: All Real Numbers
Range: All Real Numbers
Key Feature: Turning Point
Center: (0, 0)

Family Reunion

Name Equation Shape Key Feature
Linear y = x Line Slope
Quadratic y = x2 Parabola Vertex
Exponential y = 2x Curve Asymptote
Abs Value y = |x| V-Shape Sharp Vertex
Square Root y = √x Arc Endpoint
Cubic y = x3 S-Curve Turning Point
