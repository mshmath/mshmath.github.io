Algebra 1 Word Wall

Unit 1: Vocabulary & Key Concepts

Variable
A letter or symbol used to represent a value that can change or is unknown.
$3\mathbf{\color{red}x} + 5$
Coefficient
The number multiplied in front of a variable.
$\mathbf{\color{red}4}x - 7$
Constant
A number on its own that does not change. It has no variable attached.
$2x + \mathbf{\color{red}9}$
Expression
A mathematical phrase that contains numbers, variables, and operations. No equal sign.
$5x^2 - 3x + 2$
Equation
A mathematical statement that shows two expressions are equal. Has an equal sign.
$2x + 4 \mathbf{\color{red}=} 10$
Term
Parts of an expression separated by addition or subtraction signs.
$\underbrace{3x}_{\text{Term}} + \underbrace{5y}_{\text{Term}} - \underbrace{2}_{\text{Term}}$
Evaluate
To find the numerical value of an expression by substituting numbers for variables.
If $x=2$, then:
$3x \rightarrow 3(2) = \mathbf{6}$
Like Terms
Terms that have the same variables raised to the same exponents.
$\mathbf{3x}$ and $\mathbf{5x}$
$\mathbf{2y^2}$ and $\mathbf{-y^2}$
Base
The big number that is being multiplied repeatedly in a power.
$\mathbf{\color{red}5}^3$
Exponent
The small number that tells how many times to multiply the base.
$x^\mathbf{\color{red}4}$
Distributive Prop.
Multiplying a term on the outside of parentheses to every term on the inside.
$a(b+c) = \mathbf{ab + ac}$
Order of Operations
The rules to follow when simplifying expressions (GEMDAS).
1. Grouping
2. Exponents
3. M/D (L to R)
4. A/S (L to R)
Real Numbers
The set of all rational and irrational numbers. Everything on the number line.
$-2, \ 0, \ \frac{1}{2}, \ \sqrt{5}, \ \pi$
Absolute Value
The distance of a number from zero. Always positive.
$|-5| = \mathbf{5}$
Product Rule
When multiplying terms with the same base, ADD the exponents.
$x^2 \cdot x^3 = x^{\mathbf{2+3}} = x^5$
Quotient Rule
When dividing terms with the same base, SUBTRACT the exponents.
$\frac{x^5}{x^2} = x^{\mathbf{5-2}} = x^3$
Power Rule
When raising a power to another power, MULTIPLY the exponents.
$(x^2)^3 = x^{\mathbf{2 \cdot 3}} = x^6$
Negative Exponent
A negative exponent means to take the reciprocal (flip the fraction).
$x^{-2} = \frac{1}{x^2}$
Zero Exponent
Any non-zero number raised to the power of zero equals 1.
$5^0 = \mathbf{1}$
