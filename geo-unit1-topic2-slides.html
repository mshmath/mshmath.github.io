Biconditionals & Equivalence

Logic, "Iff", and Venn Diagrams

Standard G.1

Objectives & Standards

Today's Goals:

  • Identify and write Biconditional Statements.
  • Use the abbreviation Iff (if and only if).
  • Determine if two statements are Logically Equivalent.
  • Use Venn diagrams to represent logical sets.

The Biconditional Statement

Definition

A Biconditional Statement is a single statement that combines a true conditional AND its true converse.

It works "Both Ways".

p ↔ q

Read as: "p if and only if q"

Checklist: When to use "Iff"?

You can only write a biconditional if BOTH directions are True:

Forward: If p → q is True.

Backward: If q → p is True.

Result: p ↔ q

Guided Practice: Writing Biconditionals

Statement: "If points are collinear, they lie on the same line."

1. Is the Conditional True?
Yes.
2. Is the Converse True?
Yes. (If on same line, they are collinear).
3. Write the Biconditional:
Points are collinear iff they lie on the same line.

Logical Equivalence

Definition

Two statements are Logically Equivalent if they always have the same truth value.

If one is true, the other MUST be true.

The Symbol

≡

Logically Equivalent

"Statement A is equivalent to Statement B"

Ex: A ≡ B

The Equivalent Pairs

Pair #1
Conditional ≡ Contrapositive
(p → q) ≡ (~q → ~p)
Pair #2
Converse ≡ Inverse
(q → p) ≡ (~p → ~q)

Checking Truth Values

Statement: "If I live in Richmond, then I live in Virginia."

TRUE Statements

1. Conditional:
If Richmond, then Virginia. (True)

2. Contrapositive:
If NOT Virginia, then NOT Richmond. (True)

These are Equivalent! ≡

FALSE Statements

1. Converse:
If Virginia, then Richmond. (False - Norfolk)

2. Inverse:
If NOT Richmond, then NOT Virginia. (False - Norfolk)

These are Equivalent! ≡

Venn Diagrams: Conditional

"If P, then Q" (p → q)

P
Q

P is a SUBSET of Q.

If you are in the Blue circle, you are inside the Red circle.

Venn Diagrams: Biconditional

"P if and only if Q" (p ↔ q)

P = Q

P and Q are the SAME set.

They match perfectly. To be in P means you are in Q.
