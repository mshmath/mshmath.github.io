p → q

Statement A sentence that is either True or False.
Hypothesis (p) The "If" part (condition).
Conclusion (q) The "Then" part (result).
Negation (~) The opposite meaning (adds/removes "not").
Symbolic Notation Guide
Symbol Name Meaning / Example
→ Conditional "If... Then..." (Implies)
~ Negation "Not" (Opposite truth value)
∧ Conjunction "And" (Both must be true)
∨ Disjunction "Or" (One or both can be true)
↔ Biconditional "If and only if" (iff)
∴ Therefore Conclusion of an argument
The 4 Logical Statements
Form Symbol Rule
Conditional p → q Original Statement
Converse q → p FLIP the order
Inverse ~p → ~q NEGATE (add "not" to both)
Contrapositive ~q → ~p FLIP & NEGATE both
Counterexample:
A specific example that proves a statement is FALSE.
Statement: "All birds fly." → Counterexample: Penguin.
