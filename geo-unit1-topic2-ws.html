Name:

p ↔ q
BICONDITIONAL A statement combining a conditional and its converse.
KEYWORD "if and only if" (often abbreviated as iff).
RULE It is TRUE only if both the original statement and its converse are true.

Two statements are Logically Equivalent if they have the same truth value.

Levels 1-2: Biconditionals

Level 1: Identifying Biconditionals
Determine if the statement can be written as a biconditional. (Check Converse!)

Converse True? ____

Converse True? ____

Converse True? ____

Level 2: Writing Biconditionals
Combine the pair of statements into one "iff" statement.
9. If points are collinear, they lie on the same line. If points lie on the same line, they are collinear.
10. If 2x + 1 = 9, then x = 4. If x = 4, then 2x + 1 = 9.
11. If a triangle has no equal sides, it is scalene. If a triangle is scalene, it has no equal sides.
12. If x > 0, then x is positive. If x is positive, then x > 0.
13. If y = 3, then y² = 9. If y² = 9, then y = 3. (Write even if false)
14. If it is Saturday, it is the weekend. If it is the weekend, it is Saturday.

Logic Drill

Levels 3-5: Equivalence & Symbols

Level 3: Logical Equivalence
Match the statement to its logically equivalent form.
15. Which is equivalent to: "If it snows, school is closed"?
A) If school is closed, it snows.    B) If school is NOT closed, it did NOT snow.
16. Which is equivalent to: "If x = 5, then x² = 25"?
A) If x² ≠ 25, then x ≠ 5.    B) If x² = 25, then x = 5.
17. The Inverse is equivalent to the __________.
18. The Conditional is equivalent to the __________.
19. Equivalent to: "If P, then Q"?
20. Equivalent to: "If not Q, then not P"?
Level 4: Symbolic Logic
Translate words to symbols or symbols to words.
21. Symbol for "iff"
22. Words for: p ∧ q
23. Symbol for "If not p, then not q"
24. Words for: p ∨ q
25. Is (p → q) ≡ (~q → ~p)?
26. Is (q → p) ≡ (~p → ~q)?
27. Translate: ~p → q
28. Translate: ~p ↔ q
Level 5: Mixed Application & Truth Values
29. Write the Converse of: "If I eat, I am full."
30. Write the Inverse of: "If x=5, then x+2=7."
31. Determine Truth: "A square has 4 sides iff a triangle has 3 sides." (T/F)
32. Determine Truth: "2+2=5 iff 1+1=2." (T/F)
