Key Definitions
Statement A sentence that is either True or False. Represented by letters like p, q, r.
Negation (~) Means "NOT". It flips the truth value.
Ex: If p is True, ~p is False.
Conjunction (∧) Means "AND". It is "Strict".
It is True only if BOTH parts are True.
Disjunction (∨) Means "OR". It is "Lenient".
It is True if AT LEAST ONE part is True.
Logic Rules & Truth Tables
Remember:
  • p ∧ q is True only when T & T.
  • p ∨ q is False only when F & F.

Level 4: Logic Puzzles & Applications
25. Given: "If it rains, I bring an umbrella." It is NOT raining. Can you conclude I did not bring an umbrella? (Yes/No)
26. Determine Truth: "2+2=4 OR 3+3=7"
27. Determine Truth: "2+2=4 AND 3+3=7"
