Translating Words
  • Addition (+): Sum, Plus, Increased by, More than, Total.
  • Subtraction (-): Difference, Minus, Decreased by.
  • Multiplication (x): Product, Times, Twice (x2), Of.
  • Division (÷): Quotient, Split, Ratio, Half (÷2).
  • Equals (=): Is, Yields, Results in, Same as.
⚠️ SWITCH WORD ALERT! ⚠️
"Less than" and "Subtracted from"
means you must FLIP the order.
"5 less than x" becomes $x - 5$
Exponent Rules
  • Multiplying Bases? ADD the exponents.
    $x^{2} \cdot x^{3} = x^{5}$
  • Dividing Bases? SUBTRACT the exponents.
    $\frac{x^{5}}{x^{2}} = x^{3}$
  • Power to Power? MULTIPLY the exponents.
    $(x^{2})^{3} = x^{6}$
  • Negative Exponent? FLIP the fraction.
    $x^{-2} = \frac{1}{x^{2}}$
Desmos Shortcuts

Type these codes to create symbols:

  • Exponents ($x^2$): Press Shift + 6 (the ^ key)
  • Square Root ($\sqrt{x}$): Type sqrt
  • Cube Root ($\sqrt[3]{x}$): Type cbrt

Unit 2: Equations & Inequalities
Key Concepts
  • Literal Equation: An equation with mostly letters (e.g., $A=LW$). Treat the extra letters like regular numbers.
  • Inequalities Rule: If you multiply or divide by a NEGATIVE, you must FLIP the symbol!
    $-2x > 10$
    $\frac{-2x}{-2} < \frac{10}{-2}$
    $x < -5$
Graphing Inequalities Guide
Symbol Circle Bracket/Paren Meaning
$<$ or $>$ Open $\circ$ ( Parentheses ) NOT Included
$\le$ or $\ge$ Closed $\bullet$ [ Brackets ] Included
Desmos Hack: The "Line 1 / Line 2" Trick
  • How to Solve Equations:
    1. Type Left Side in Line 1 ($y = -2x+5$).
    2. Type Right Side in Line 2 ($y = 15$).
    3. Click the intersection dot. The X-value is your answer.
  • Inequality Symbols:
    $\le$ (Less or Equal): Type <=
    $\ge$ (Greater or Equal): Type >=

Unit 3: Functions & Graphs
Key Concepts
  • Slope ($m$): Rise over Run. How steep the line is.
  • Y-Intercept ($b$): Where the line hits the vertical Y-axis. The "Start".
  • Equation Forms:
    Graphing: $y = \mathbf{m}x + \mathbf{b}$
Desmos Hacks
  • Find Slope & Y-Int: Type the equation (e.g., $3x+2y=12$). Look at the graph. Click the gray dots on the axes.
  • Find Slope from Points:
    1. Add a table (+ Table).
    2. Enter the points.
    3. Select Linear Regression from the dropdown menu (or type y1 ~ mx1 + b).

Unit 4: Systems of Equations
Key Concepts
  • System: Two or more equations graphed together.
  • Solution: The point $(x, y)$ where lines INTERSECT.
  • No Solution: Parallel lines (never cross).
  • Infinite Solutions: Same line (on top of each other).
Desmos Hack: Just Graph It
  • 1. Type Equation 1 in Box 1.
  • 2. Type Equation 2 in Box 2.
  • 3. Click the gray dot where they cross. That coordinate is your answer.

Unit 5: Polynomials & Characteristics
Key Concepts (Definitions)
  • Vertex (Max/Min): The highest or lowest point of the graph.
  • Zeros / Roots / X-Intercepts: Where the graph crosses the X-axis ($y=0$).
  • Y-Intercept: Where the graph crosses the Y-axis ($x=0$).
  • Turning Point: Where a curve changes direction (bumps/dips in a Cubic function).
  • Domain: How far Left and Right the graph goes (X-values).
  • Range: How far Down and Up the graph goes (Y-values).
Desmos Hack: Finding Characteristics
  • Step 1: Type the quadratic equation ($y = x^2 - 5x + 6$).
  • Step 2: Look at the graph.
  • Step 3: Click the gray dots. Desmos will instantly highlight the Roots, Vertex, and Y-intercept coordinates for you.

Unit 6: Statistics
Function Families (Visuals)
  • Linear ($y=mx+b$): A straight line.
  • Quadratic ($y=ax^2$): A "U" shape (Parabola).
  • Exponential ($y=ab^x$): A "J" curve (Hockey Stick).
Desmos Hack: Stats Codes

Option 1: Type numbers directly: mean(1, 5, 8)

Option 2: Make a List!
1. Type L = [1, 5, 8, 10] (Use square brackets).
2. Then use L in the codes below.

  • Mean (Average): Type mean(L)
  • Median (Middle): Type median(L)
  • Standard Deviation: Type stdev(L)
  • Range: Type max(L) - min(L)
  • Stats (5-Number Summary): Type stats(L) to get Min, Q1, Median, Q3, Max.
  • IQR (Interquartile Range): Find Q3 and Q1 using stats(L), then subtract: $Q3 - Q1$.
  • Regression (The Menu):
    1. Add a Table ($+$).
    2. Click the Magnifying Glass (Zoom Fit).
    3. Select the Function Type from the dropdown (Linear, Quadratic, Exponential).
