The "If-Then" Statement

Conditional Statement (p → q)

A logical statement with two parts: a condition and a result.

If it rains, then I will get wet.
Hypothesis (p)

The part following "If".
(The Cause)

Do NOT include the word "If".

Conclusion (q)

The part following "Then".
(The Effect)

Do NOT include the word "Then".

Guided Practice: Identify Parts

"If an animal is a poodle, then it is a dog."

Identify the Hypothesis (p):
an animal is a poodle
Identify the Conclusion (q):
it is a dog

Logic Symbols

Key Symbols

p : Hypothesis

q : Conclusion

→ : Implies (If...then)

∼ : Negation (Not)

Connectors

∧ : And (Conjunction)

∨ : Or (Disjunction)

↔ : Iff (Biconditional)

∴ : Therefore

The 4 Logical Statements

1. Conditional
p → q

If p, then q.

2. Converse
q → p

FLIP IT.

If q, then p.

3. Inverse
∼p → ∼q

NEGATE IT.

If NOT p, then NOT q.

4. Contrapositive
∼q → ∼p

FLIP & NEGATE.

If NOT q, then NOT p.

Guided Practice: Writing Forms

Original: "If it is raining, then the grass is wet."

Converse (Flip):
If the grass is wet, then it is raining.
Inverse (Negate):
If it is NOT raining, then the grass is NOT wet.
Contrapositive (Both):
If the grass is NOT wet, then it is NOT raining.

Proving it False: Counterexamples

Definition

A specific example that shows a statement is FALSE.

You only need ONE counterexample to prove a statement false.

Statement: "All birds can fly."

Counterexample:

A Penguin (It is a bird, but it cannot fly).

Logically Equivalent

Who Tells The Truth?

Some statements always have the same Truth Value.

Pair 1

Conditional ↔ Contrapositive

If the original is True, the Contrapositive is also True!

Pair 2

Converse ↔ Inverse

These two are logically linked to each other.

Biconditional Statements

p ↔ q

"p if and only if q"

  • Use this when BOTH the Conditional AND Converse are true.
  • Definitions are always biconditional.
  • Example: "A triangle is equilateral if and only if all 3 sides are equal."
