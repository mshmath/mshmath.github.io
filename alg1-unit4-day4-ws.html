Worked Example

Graph the system and identify a solution: $y \ge x - 2$ and $y < -x + 4$.

  1. Step 1 (Graph the first inequality): For $y \ge x-2$, the line is solid because of "or equal to." We shade above because of "greater than."
  2. Step 2 (Graph the second inequality): For $y < -x+4$, the line is dotted because it's just "less than." We shade below because of "less than."
  3. Step 3 (Find the Solution Area): The solution is the region on the graph where the two shaded areas overlap.
  4. Step 4 (Identify One Solution): Pick any point from the overlapping area. For example, the point $(0,0)$ is in the solution region. Let's check it:
    $0 \ge 0-2 \Rightarrow 0 \ge -2$ (True)
    $0 < -0+4 \Rightarrow 0 < 4$ (True)
    Since it works for both, $(0,0)$ is a valid solution.
