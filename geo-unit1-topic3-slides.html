Compound Statements

Combining two or more statements.

Conjunction (AND)
p ∧ q

"P AND Q"

Think: Intersection

Disjunction (OR)
p ∨ q

"P OR Q"

Think: Union

The Conjunction (∧)

The Strict Parent

A conjunction is TRUE only if BOTH parts are true.

"I want a burger AND fries."

  • Burger only? FALSE
  • Fries only? FALSE
  • Both? TRUE

The Disjunction (∨)

The Lenient Parent

A disjunction is TRUE if AT LEAST ONE part is true.

"I want a burger OR fries."

  • Burger only? TRUE
  • Fries only? TRUE
  • Neither? FALSE

Intro to Truth Tables

A table showing all possible truth values for a statement.

Setup (2 Variables)

Start with standard columns for p and q.

p q
TT
TF
FT
FF

T = True

F = False

We use these inputs to calculate complex logic.

Truth Table: Negation (~)

Negation flips the truth value.

p ~p
T F
F T

If p is True, Not p is False.

Truth Table: Conjunction (∧)

Remember: AND is Strict. Needs T-T.

p q p ∧ q
TT T
TF F
FT F
FF F

Truth Table: Disjunction (∨)

Remember: OR is Lenient. Just needs one T.

p q p ∨ q
TT T
TF T
FT T
FF F

Guided Practice: Translating

p: Math is fun.    q: Logic is hard.

1. Translate: p ∧ ~q
"Math is fun AND logic is NOT hard."
2. Translate: ~p ∨ q
"Math is NOT fun OR logic is hard."

Guided Practice: Truth Values

p: 2 + 2 = 4 (T)    q: 5 > 10 (F)

1. p ∧ q (T AND F)
FALSE
2. p ∨ q (T OR F)
TRUE

Independent Practice: Level 1

Write the Symbols.

1) "Today is Monday AND it is sunny."
2) "I will eat pizza OR tacos."
3) "It is NOT raining."

Independent Practice: Level 2

Determine Truth Values (p=True, q=False)

1) p ∨ q
2) p ∧ q
3) ~p ∨ ~q (Hint: Translate first)

Independent Practice: Level 3

Complex Logic (p=T, q=F, r=T)

1) Determine Truth: (p ∧ q) ∨ r
(Hint: Solve parentheses first)
2) Determine Truth: ~p ∧ (q ∨ r)
3) Is (p ∨ q) true? Why?
