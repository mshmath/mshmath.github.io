TURN AROUND RULE:
"Less than" or "Subtracted from"
5 less than x → x - 5
2. Evaluating Expressions
Substitute: Replace variable with number.
GOLDEN RULE:
Use Parentheses when substituting!
x² where x = -3 → (-3)² = 9
PEMDAS: (1) Parentheses (2) Exponents (3) Mult/Div (4) Add/Sub.
3. Laws of Exponents
  • Product: Multiplying same base? ADD exponents.
    (xᵃ • xᵇ = xᵃ⁺ᵇ)
  • Quotient: Dividing same base? SUBTRACT exponents.
    xᵃxᵇ
    = xᵃ⁻ᵇ
  • Power: Raising a power? MULTIPLY exponents.
    ((xᵃ)ᵇ = xᵃ•ᵇ)
  • Zero: Zero power? It equals 1.
    (x⁰ = 1)
  • Negative: Negative power? FLIP the fraction.
    x⁻ᵃ =
    1xᵃ
4. Simplifying Expressions
Like Terms: Same Variable AND Same Exponent.
3x + 5x = 8x (Add Coefficients)
Distributive Property:
Multiply outside term by EVERYTHING inside.
2(x + 4) → 2x + 8
SIMPLIFICATION ORDER:
1. Distribute first to clear any parentheses.
2. Then Combine Like Terms.
