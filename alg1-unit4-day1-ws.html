Worked Example

Find the solution to the system: $y = -x + 3$ and $y = 2x$.

Method 1: By Graphing

  1. Graph both lines on Desmos.
  2. Find the point where they cross.
  3. Click the point to see the coordinates. The solution is $(1, 2)$.

Method 2: By Hand (Algebra)

  1. Step 1: Since both equations equal $y$, set them equal to each other: $-x + 3 = 2x$.
  2. Step 2: Solve for $x$. Add $x$ to both sides to get $3 = 3x$. Divide by $3$ to get $x=1$.
  3. Step 3: Plug $x=1$ into one of the original equations to find $y$. Let's use the second one: $y = 2(1)$, so $y=2$.
  4. Step 4: The solution is the point $(1, 2)$.
